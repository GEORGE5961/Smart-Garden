### This is a React component which serves as a heatmap.

It is based on Bizcharts supported by Alibaba.

## How to use

In bash:

```
npm install bizchart
```

In App.js file:

```
import { Chart, Geom, Tooltip, Legend,Guide } from 'bizcharts';

```
